# Django Quiz Web App

Welcome to the Django Quiz Web App! This application hosts quizzes covering Python, JavaScript, HTML, and CSS.

## Features

- Leaderboard: View top scorers.
- Recent Quizzes: Check out the latest quizzes added.
- About Me: Learn more about the developer.

## Usage

- Sign up or log in to start taking quizzes.
- Explore the leaderboard to see top scorers.
- Check out recent quizzes to stay updated.
- Visit the "About Me" section to learn more about the developer.
- Manage your user profile settings

## Technologies Used

- Django
- HTML/CSS (Tailwind)
- JavaScript
